# MemoryBread client brand assets

- `memorybread-app-icon.svg`: light application and browser-icon master.
- `memorybread-mark.svg`: transparent brand mark used by client navigation.
- `raster/`: alpha-safe PNG exports from the transparent mark.

Brand colors are warm ivory `#FFF4DF`, deep cocoa `#43281F`, and baking orange `#F0782D`.
