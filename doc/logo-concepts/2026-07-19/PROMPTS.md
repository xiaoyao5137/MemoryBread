# MemoryBread logo concept prompts

Built-in `imagegen` was used. The existing Tauri icon was supplied only as a brand reference; no existing asset was edited or replaced.

## Candidate A — Friendly Continuity

```text
Use case: logo-brand
Asset type: candidate app icon and in-app brand mark for the Chinese desktop AI assistant “记忆面包 / MemoryBread”
Input images: the current app icon, used only as a brand reference for the warm bread identity and friendly tone; create an original redesign, do not trace it.
Primary request: Design one centered, instantly recognizable toast character icon. Use a simplified front-facing bread slice with a very subtle smiling expression and one distinctive memory motif: a single warm-orange ribbon folds once across the bread interior like a bookmark/history trail. The character should feel calm, trustworthy, useful, and quietly intelligent, not childish.
Style/medium: polished vector-inspired macOS app icon, mostly flat colors with restrained soft depth; custom geometric shapes; premium independent software quality.
Composition/framing: square 1:1; one centered mark; generous safe margin; strong silhouette readable at 16px; full-bleed softly rounded square background.
Color palette: warm off-white #FFF7E8, baking orange #E9792B, toast gold #F2B45A, deep cocoa #4B2E24; high contrast.
Constraints: no text, letters, slogan, brain, circuitry, sparkles, mockup, device, hands, or watermark; avoid glossy 3D, photorealism, blue, purple, and unnecessary detail.
```

## Candidate B — Memory Layers

```text
Use case: logo-brand
Asset type: candidate app icon and in-app brand mark for the Chinese desktop AI assistant “记忆面包 / MemoryBread”
Input images: the current app icon, used only as a brand reference for the warm bread identity; create an original redesign, do not trace it.
Primary request: Create a compact symbol made from exactly three overlapping bread-slice layers, slightly offset upward and right like a tidy stack of saved cards or accumulated memories. The front slice has one small centered bookmark-shaped cutout in negative space. No face. The symbol should communicate knowledge accumulating over time, local archives, and reliable recall without using a brain or circuitry.
Style/medium: premium custom vector-inspired macOS app icon; flat geometric forms with restrained depth only from clean layered offsets; calm, modern, professional.
Composition/framing: square 1:1; centered compact mark; generous safe margin; strong silhouette readable at 16px; full-bleed deep cocoa softly rounded square background.
Color palette: deep cocoa #3B241C background, warm off-white #FFF4DF front slice, toast gold #F2B45A middle layer, baking orange #E9792B back layer.
Constraints: exactly three bread layers; no text, letters, face, brain, circuitry, sparkles, mockup, device, hands, watermark, blue, purple, glossy 3D, photorealism, crumbs, or complex shadows.
```

## Candidate C — Bread Loop

```text
Use case: logo-brand
Asset type: candidate app icon and in-app logo mark for MemoryBread
Input images: the first Bread Loop draft as the edit target.
Primary request: Change only the internal geometry so it cannot be mistaken for a padlock, computer mouse, stethoscope, headphones, or moustache. Remove the dangling central circular loop and symmetrical inner arches. Keep a bold bread-slice outer silhouette. Inside it, use one simple asymmetric continuous memory path: start near the lower-left interior, rise in one broad calm curve through the center, and finish as a short orange folded-bookmark segment at the upper-right interior. The negative space should feel like a saved path through time, not a literal letter or common tech symbol.
Style/medium: minimal custom monoline vector logo, flat geometric forms, optically balanced.
Constraints: keep the warm off-white background, deep cocoa and baking-orange palette, square composition, stroke weight, safe margin, and rounded app-icon tile; no face, text, brain, circuitry, infinity sign, central circle, teardrop, symmetrical handle shapes, or watermark; readable at 16px.
```

## Production note

These are raster concept previews for selection. After a direction is chosen, redraw the selected geometry as SVG, tune the 16/32/64px optical sizes, export transparent PNG and ICNS/ICO sets, and only then consider replacing application assets in a separate approved change.
